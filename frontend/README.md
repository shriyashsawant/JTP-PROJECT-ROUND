# AuraMatch AI - Frontend

Next.js 16 (App Router) web client for AuraMatch AI. This app is not run standalone in development - it's one of three services in the root [`docker-compose.yml`](../docker-compose.yml); see the root [README.md](../README.md) for the actual `docker compose up` workflow.

For the page-by-page architecture, state management, and API integration details, see [`documentation/FRONTEND_ARCHITECTURE.md`](../documentation/FRONTEND_ARCHITECTURE.md).
